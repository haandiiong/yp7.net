## 测评原则

yp7.net 的机场测评更关注真实使用体验，而不是单次测速截图。机场服务会受用户地区、运营商、晚高峰、线路维护、节点负载和平台策略影响，因此任何结论都需要持续复测。

## 核心指标

| 指标 | 观察内容 | 说明 |
|---|---|---|
| 晚高峰稳定性 | 20:00-23:00 网页、视频、ChatGPT 是否稳定 | 比白天测速更重要 |
| 延迟 | Clash 或客户端延迟测试 | 只做初筛，低延迟不等于高速 |
| 下载速度 | Speedtest、实际下载、视频缓冲 | 观察真实吞吐 |
| 流媒体 | YouTube 4K、Netflix、Disney+、TikTok | 解锁能力会变化 |
| ChatGPT | 登录、连续对话、断连情况 | 适合办公用户重点测试 |
| 客户端 | 专属客户端、Clash、Shadowrocket、Mihomo、sing-box | 决定上手成本和灵活性 |
| 价格与流量 | 最低月付、月流量、试用、不限时套餐 | 不只看最低价 |
| 风险 | 官网、售后、退款、支付、历史异常 | 决定是否适合长期付费 |

## 推荐测试流程

1. 先看 [全量机场榜单](/rankings/all/) 缩小候选范围。
2. 根据场景查看 [稳定机场榜](/rankings/stable/)、[Clash机场榜](/rankings/clash/)、[ChatGPT机场榜](/rankings/chatgpt/) 或 [流媒体机场榜](/rankings/streaming/)。
3. 阅读单个机场测评页，确认价格、试用、客户端和风险提示。
4. 购买月付、试用或短周期套餐。
5. 在自己的常用设备、地区、运营商和晚高峰时间段测试。
6. 确认稳定后再考虑更长周期套餐。

## 不采用的判断方式

- 不只按最低价格排序。
- 不只按单次 Speedtest 结果排序。
- 不把服务商宣传文案当作稳定性证据。
- 不承诺任何机场长期永久可用。

## 常见问题

### 为什么晚高峰测试最重要？

晚高峰用户最多，更容易暴露节点拥堵、线路绕路、售后维护和实际稳定性问题。

### 为什么测评结论会变化？

机场节点、套餐、优惠码、官网入口、支付方式和流媒体策略都会变化，因此测评需要持续更新。
