## 🌐 XXYUN基础信息

XXYUN是一类提供多地区节点的订阅服务，采用 **BGP多线中转架构**。

主要特点：

- 三网优化（电信 / 联通 / 移动）  
- 多地区节点（香港 / 日本 / 新加坡）  
- 支持主流协议（Trojan / V2Ray / Shadowsocks）  

👉 BGP线路在兼容性方面表现较好，但稳定性通常略低于IEPL专线。
