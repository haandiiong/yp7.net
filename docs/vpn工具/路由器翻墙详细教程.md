## 什么是路由器翻墙

路由器翻墙是指在 **无线路由器上配置VPN或代理协议**，让所有连接到该WiFi网络的设备自动通过VPN访问互联网。

简单来说：


设备 → 路由器 → VPN服务器 → 国际互联网


这样可以让：

- 手机翻墙
- 电脑翻墙
- 智能电视翻墙
- 游戏机翻墙
- IoT设备翻墙

全部自动完成。

相比单设备VPN，**路由器VPN更加方便和稳定**。

## 路由器VPN翻墙的基本要求

在开始配置之前，需要满足以下条件。

### 1 路由器支持VPN

常见支持方式：

- 原生VPN客户端
- OpenWRT固件
- DD-WRT固件
- Merlin固件

部分品牌默认支持：

- 华硕
- Netgear
- Linksys

### 2 VPN服务支持路由器

推荐选择支持：

- OpenVPN
- WireGuard
- V2Ray
- Trojan

协议的机场或VPN。

### 3 路由器性能足够

VPN加密需要CPU计算能力。

建议：

| 使用人数 | 路由器CPU |
|---|---|
1-3人 | 双核 |
3-6人 | 四核 |
家庭多设备 | WiFi6路由器 |

## 小米路由器翻墙教程

小米路由器系统基于OpenWRT。

- 小米3
- AC2100
- AX3600
- AX6000

刷机步骤：

1 开启SSH  
2 上传固件  
3 刷OpenWRT  
4 安装OpenVPN

安装VPN：


opkg update
opkg install openvpn-openssl
opkg install luci-app-openvpn
