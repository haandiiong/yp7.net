## 一、TikTok是什么？

TikTok是由字节跳动（ByteDance）推出的国际短视频平台，是抖音的海外版本。

### ✔ TikTok核心优势

- 🌍 全球用户（欧美 / 东南亚）
- 🤖 算法推荐（新号也能爆）
- 🎬 创作门槛低（手机即可）
- 💰 可直接变现（带货 / 广告 / 引流）

👉 简单理解：
**TikTok = 国际版抖音 + 全球流量池 + 赚钱工具**
