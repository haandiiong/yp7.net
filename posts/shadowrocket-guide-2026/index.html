Shadowrocket（小火箭）是目前 iOS端最主流的科学上网工具之一。
支持SS、V2Ray、Trojan等协议，具备规则分流、广告拦截等功能。

本文按常见使用流程整理：

  • Shadowrocket下载安装（App Store）
  • 小火箭订阅导入方法
  • 节点选择与连接
  • 分流规则设置
  • 常见问题解决

👉 适用于：iPhone / iPad / iOS用户

一、Shadowrocket是什么？（新手必读）

Shadowrocket 是一款 iOS 网络代理工具，主要用于：

  • 科学上网（访问Google、YouTube等）
  • 提升访问速度
  • 保护隐私

支持协议

  • Shadowsocks（SS）
  • V2Ray（VMess / VLESS）
  • Trojan
  • Socks5

核心优势

  • 一键导入订阅
  • 智能分流（国内直连 / 国外代理）
  • 广告拦截
  • 高稳定低延迟

二、Shadowrocket下载方法

⚠️ 为什么App Store搜不到？

Shadowrocket 已从中国区下架，需要使用：

  • 美区 Apple ID
  • 港区 Apple ID

📥 正确下载安装步骤（完整版）

1️⃣ 打开 App Store
2️⃣ 点击头像 → 退出当前账号
3️⃣ 登录海外 Apple ID
4️⃣ 搜索：Shadowrocket或直接下载👇
5️⃣ 点击下载（约 $2.99）
6️⃣ 安装完成后打开

如果没有海外Apple ID可选择共享海外 Apple ID 获取https://yh.vip9.pro

⚠️ 重要提示

  • 下载后立即退出共享账号
  • 避免开启iCloud
  • 若验证手机号 → 更换账号
  • 若提示安全问题 → 登录步骤错误

三、Shadowrocket怎么用？（核心步骤）

第一步：获取订阅（必须）

Shadowrocket本身不能直接使用，需要：

👉 购买机场订阅

推荐阅读：
👉 机场推荐

第二步：导入订阅（最关键）

1️⃣ 打开 Shadowrocket
2️⃣ 点击右上角 "+"

3️⃣ 设置：

  • 类型：Subscribe
  • URL：粘贴订阅链接

4️⃣ 点击保存

✅ 自动导入节点

第三步：选择节点

常见节点：

  • 美国 🇺🇸
  • 日本 🇯🇵
  • 新加坡 🇸🇬

👉 建议：

  • 日常：低延迟节点
  • 视频：高速节点

第四步：开启连接

1️⃣ 打开开关
2️⃣ 授权 VPN

成功标志：

✅ 状态栏出现 VPN 图标

四、Shadowrocket配置详解（进阶）

📌 路由模式

路径：

👉 设置 → 路由

推荐：

  • 配置（默认）
  • 代理（全局）

📌 分流规则说明

  • 国内网站 → 直连
  • 国外网站 → 代理

👉 优点：

  • 更快
  • 更省流量

📌 节点测速

👉 首页 → 连通性测试

选择：

  • 延迟最低
  • 常用稳定节点

五、如何判断是否成功？

打开：

👉 https://www.google.com

如果能访问：

✅ 说明成功

六、常见问题

❓ Shadowrocket下载不了怎么办？

原因：

  • 未使用海外Apple ID

解决：

  • 切换美区/港区账号

❓ 订阅导入失败？

检查：

  • 链接是否过期
  • 是否复制完整
  • 网络是否正常

❓ 无法连接？

解决：

  • 切换节点
  • 更新订阅
  • 检查网络

❓ 速度慢？

优化：

  • 更换节点
  • 使用测速
  • 更换机场

七、进阶优化技巧（提高体验）

  • DNS：1.1.1.1 / 8.8.8.8
  • 开启广告拦截
  • 使用优质规则集
  • 定期更新订阅

八、总结

Shadowrocket使用核心只有三步：

1️⃣ 下载（海外Apple ID）
2️⃣ 添加订阅
3️⃣ 一键连接

对于 iOS 用户来说，小火箭仍是常见的代理客户端之一，但实际体验取决于订阅质量、节点状态和本地网络。
