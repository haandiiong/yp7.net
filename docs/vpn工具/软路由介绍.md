## 一、什么是软路由？

**软路由（Soft Router）**，指的是使用通用硬件（如 x86 主机、迷你主机、旧电脑等），安装专业路由系统，实现高性能网络管理的一种方案。

与传统路由器不同，软路由的核心在于：

> **硬件 + 软件解耦 = 无限扩展能力**

你可以自由更换系统、安装插件、调整策略，几乎没有限制。

---

## 二、软路由 vs 普通路由器（核心区别）

| 对比项 | 普通路由器 | 软路由 |
|------|-----------|--------|
| 硬件 | 固定 | 自由搭配 |
| 系统 | 厂商封闭 | 开源/商业系统 |
| 插件扩展 | 很少 | 极其丰富 |
| 科学上网能力 | 弱 | 强 |
| 可玩性 | 低 | 极高 |

👉 简单一句话总结：

> **普通路由器是“家电”，软路由是“电脑”。**

---

## 三、哪些人适合用软路由？

软路由并不是所有人都需要，以下人群更适合：

- ✔ 经常使用科学上网（Clash / VPN）
- ✔ 家庭设备较多（手机 / 平板 / TV / NAS）
- ✔ 需要分流策略（国内直连 / 国外代理）
- ✔ 对延迟、带宽有较高要求
- ✔ 喜欢折腾、追求自由配置

如果你只是刷视频、看直播，其实普通路由器就够用了。

## 六、OpenWrt 安装教程（新手步骤）

### 安装流程：

1. 下载 OpenWrt x86 镜像  
2. 使用 Rufus / balenaEtcher 写入U盘

> 推荐使用Rufus下载地址 [https://github.com/pbatard/rufus/releases/download/v4.13/rufus-4.13.exe](https://github.com/pbatard/rufus/releases/download/v4.13/rufus-4.13.exe)
3. 设置设备从U盘启动  
4. 安装系统到硬盘  
5. 配置 LAN / WAN  
6. 浏览器进入后台管理  

完成后，即可进入插件配置阶段。

不难，按教程操作，1小时即可完成基础搭建。

10W–20W，非常省电。

不需要，家庭宽带即可。

N100 级别基本可以跑满千兆。

约 3～8 元，成本极低。

### 需要几个网口？
至少 2 个，建议 4 个更灵活。

## 十、软路由 vs 旁路由（新手必看）

很多人会混淆：

- **软路由**：主路由（核心网络设备）
- **旁路由**：辅助代理设备

👉 简单理解：

> 旁路由 = 不改变原网络结构的“外挂加速器”
